ネコサーチAPI

ネコサーチの最新投稿を外部サービスでも手軽に扱えるように、JSON APIをご用意しました。
リクエストされたURLのクエリ条件に見合う、最新10件の情報を返します。
ぜひご利用ください。

使い方

リクエストURLは下記になります。

http://api.neko-search.com/v1/?type=(search|protect)&area=(都道府県名)

type

'search'(探されている猫の情報)、もしくは'protect'(保護されている猫)を記述してください。
type自体の指定がない場合は両方を含むデータが返されます。

area

都道府県名を記述してください。
名称は○○県や○○府など、都道府県の文字まで記述してください。
area自体の記述がない場合は全都道府県を含むデータが返されます。

受け取れるデータ

  • site_url : その情報が記載されているネコサーチ上のページURL
  • name : 猫の名前(保護されている猫の場合、空文字)
  • area : 都道府県 市区郡 町村
  • best_feature : 特徴
  • body_color : 体毛の色(記載がない場合、空文字)
  • eye_color : 瞳の色(記載がない場合、空文字)
  • pict_url : 投稿された画像のURL
